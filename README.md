# VZ Follow

クライアント側 Fabric Mod（Minecraft **1.21.11**）。
最寄りプレイヤーの自動追尾と、OptiFine風のズームを備えています。

## 機能
### 追尾（R キー）
- **R キー**（変更可）で追尾の ON/OFF を切り替え
- 最も近いプレイヤーへ向かって自動で歩行／離れていればダッシュで追従
- **安全モード**: 2ブロック以上の落下・溶岩・水・火・マグマブロックを検知して停止
- 1ブロックの段差は自動ジャンプ
- 移動はバニラのキー入力経由なので、通常の当たり判定・物理がそのまま適用されます
- インベントリ／チャット／メニューを開いている間は移動しません

### ズーム（C キー）
- **C キー**を**押している間**だけズーム（OptiFineのズームと同じ操作感）
- 視野角(FOV)を約1/4に縮めて拡大表示し、離すと元のFOVに戻ります
- ズーム中はマウス感度を自動で下げて狙いやすく（離すと元に戻ります）

> クライアント専用Modです。サーバー側へ入れる必要はありません（マルチサーバーでもそのまま使えます）。
> 各プレイヤーが自分のクライアントに入れて使います。

## ダウンロード
最新版は [Releases](../../releases) からダウンロードしてください（`vzfollow-x.y.z.jar`）。

## 導入方法
1. **Fabric Loader**（Minecraft 1.21.11 用）を導入する
2. **Fabric API** を `mods` フォルダに入れる
3. `vzfollow-x.y.z.jar` を `mods` フォルダに入れる
4. ゲームを起動し、設定 → キー設定 で「追尾の切り替え」（初期: **R**）と「ズーム」（初期: **C**）を確認

必要環境:
- Minecraft 1.21.11
- Fabric Loader 0.19.0 以上
- Fabric API
- Java 21 以上

## 自分でビルドする
```sh
# Linux / macOS
./gradlew build

# Windows (PowerShell)
.\gradlew.bat build
```
生成物は `build/libs/vzfollow-x.y.z.jar` に出力されます（`-sources.jar` ではない方を使ってください）。
